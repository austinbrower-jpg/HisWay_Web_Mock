# Media Owner Review Guide

This guide is for the owner review session only. It explains which images are in the current concept, where they appear, and what still needs a decision.

- All four concept images below are still `publicUseApproved: false` in the repository.
- The concept site stays in `noindex, nofollow` mode.
- Nothing in this guide approves a public launch by itself.

## Approval Levels

- **Approved for concept use**: okay only inside the private noindex mockup.
- **Approved for public website use**: okay on the live website in normal public pages such as homepage, About, or service pages.
- **Approved for portfolio or case-study use**: okay only when the work is being presented as a specific customer project and the facts and permission are documented.
